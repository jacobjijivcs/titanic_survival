# Titanic Survival

This project analyzes Titanic passenger data and builds models to predict survival.

Contents:
- `titanic_dataset.csv` — dataset used for analysis
- `titanic_humanized.py` — analysis and modeling script
- `fig1_who_was_on_board.png`, `fig2_age_and_fare_patterns.png`, `fig3_correlation_map.png`, `fig4_model_verdict.png` — figures

How to run:
1. Create and activate a Python virtual environment.
2. Install required packages (e.g., pandas, matplotlib, scikit-learn).
3. Run `titanic_humanized.py` to reproduce analysis.
